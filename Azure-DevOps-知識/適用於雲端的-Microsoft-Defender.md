
# DevOps 安全性

DevOps 安全性可處理 DevOps 與目前威脅環境的交集。它提供端對端安全性，包括程式碼和程式碼管理系統的可見度，以及有助於防止、偵測及回應目前威脅的安全性功能。藉由轉移剩餘的雲端安全性，雲端應用程式生命週期各個階段 (開發、組建及作業) 的風險可以提早處理。
