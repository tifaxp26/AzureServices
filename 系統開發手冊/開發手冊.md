# 專案開發說明 
## Walsin.DataPlatform.Web
### Controller
- 命名: 名詞，以專案名稱、用途名稱為主
- 後綴: Async
- camel case 規範

如: L5DIPController

### Action
- 命名: 動詞，以目的為主
- 前綴: Read
- 後綴: Async
- camel case 規範
- 必備項目: 使用非同步寫法(async, awiat)，套用swagger、Log、Exception 特徵屬性

如: ReadItem1Async
EX:
Read NOTESV_DIP1_FORCE_RELEASE_SC Async
Read NotesvDip1ForceReleaseSc Async

## Walsin.DataPlatform.Service
### 必要項目: 
服務介面、服務實作，定義資料模型
### 資料模型定義: 
- 屬性以英文、camel case規範



## Walsin.DataPlatform.Core
### 必要項目: 
符合命名空間 及 靜態服務，讓其他服務共用，有效加速開發
