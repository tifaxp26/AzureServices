# 可通訊的安全性群組
**Mail Enabled Security Group:**
 
透過PowerShell 登入擁有者的郵箱地址，然後執行指令刪除或添加成員:

```
Connect-exchangeonline

Remove-DistributionGroupMember -Identity "mailenabledsecurity1@tingcong.cloudns.cl" -Member "tt1@tingcong.cloudns.cl"

get-distributiongroupmember -Identity "mailenabledsecurity1@tingcong.cloudns.cl" |fl primarysmtpaddress, displayname

Add-DistributionGroupMember -Identity "mailenabledsecurity1@tingcong.cloudns.cl" -Member "tt1@tingcong.cloudns.cl"
```
